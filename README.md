# Excel-Dashboard
Air quality analysis
•	Developed an interactive Excel dashboard to analyze air-quality metrics using Pivot Tables, Pivot Charts, and Slicers for dynamic data exploration.
•	Cleaned, transformed, and standardized large environmental datasets to ensure accurate trend analysis and regional comparison of pollutant levels.
•	Identified key air-quality patterns, high-risk zones, and measure-wise insights to support data-driven environmental assessment.
